# ERC-7730 Transaction Intent Snap - MetaMask Submission Goal

## Final Goal
Submit this snap to MetaMask Snaps store so users can translate blockchain transactions into human-readable intents using ERC-7730 metadata.

## What it does
- Fetches ERC-7730 metadata for contract interactions
- Converts raw transaction data into clear, understandable descriptions
- Shows users what their transaction actually does before signing
